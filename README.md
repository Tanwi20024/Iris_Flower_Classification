# 🌸 Iris Flower Classifier

A machine learning web app that classifies iris flowers into 3 species
— Setosa, Versicolor, and Virginica.

## 🚀 Live Demo
[Click here to try the app]([https://your-app-link.streamlit.app](https://irisflowerclassification-project5.streamlit.app))

## 🛠 Tech Stack
Python · scikit-learn · Streamlit · pandas · matplotlib · seaborn

## ▶️ Run Locally
pip install -r requirements.txt
streamlit run app.py
